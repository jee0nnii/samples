GridBase SetEditOptions

Description

그리드 편집 방식과 관련된 정보들을 재설정한다. EditOptions이 설정 모델이다.

Syntax

function setEditOptions(options)

Parameters

options

Type: Object

EditOptions 모델과 같은 설정 정보. EditOptions 중 변경하고자 하는 값들만 전달하면 된다.

Return value

None.

Example

grid.setEditOptions({
deletable: true,
readOnly: false,
...
});
