LocalDataProvider onRowStatesCleared

Description

DataProvider.clearRowStates를 호출하여 rowState가 deleted이거나 createAndDeleted인 데이터를 삭제하고 rowState를 초기화 한 후 호출된다.

Syntax

function onRowStatesCleared(provider)

Arguments

provider

Type: LocalDataProvider

LocalDataProvider object.

Return

None.

Example

dataProvider.onRowStatesCleared = function (provider) {
console.log("RowStatesCleared");
};
