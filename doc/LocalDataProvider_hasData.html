LocalDataProvider hasData

Description

지정한 데이터행이 실제 값이 들어 있는 행이면 true를 리턴한다. setRowCount로 생성된 가상 행 상태이면 false를 리턴한다.

Syntax

function hasData(row)

Parameters

row

Type: number

행 번호

Return value

Type: boolean.

값이 있는 행이면 true. 가상 행이면 false.

Example

if (!provider.hasData(10)) {
alsert('10 row is virtual');
}
