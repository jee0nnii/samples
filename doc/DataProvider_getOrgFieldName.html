DataProvider getOrgFieldName

Description

매개변수로 지정된 순서에 해당하는 데이터필드의 이름을 가져온다. 필드명은 대문자로 변경되어 있다. 이 함수는 지정 당시의 필드명을 리턴한다.

Syntax

function getOrgFieldName(fieldIndex)

Parameters

Type: number

필드 인덱스.

Return value

Type: string

필드 이름.

Example

var field = 1;
var fieldName = provider.getOrgFieldName(field);
