GridBase onMenuItemClicked

Description

addPopupMenu로 추가한 팝업메뉴 항목이 클릭됐을 때 호출된다.

Syntax

function onMenuItemClicked (grid, data)

Arguments

grid

Type: GridView

GridView 컨트롤

Return

None.

Example

grid.onMenuItemClicked = function (grid, data) {
var s = data.label + (data.checked ? " checked" : "");
if (data.tag)
s += "\n" + "tag: " + data.tag;
alert(s);
};
