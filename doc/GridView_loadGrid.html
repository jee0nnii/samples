GridView loadGrid

Description

그리드의 필드 및 컬럼 정보 등 모든 설정 정보 및 데이터를 한번에 설정한다.

Syntax

function getPage()

Parameters

Type: Objects

필드 및 컬럼 정보 등 모든 설정 정보 및 데이터

Return value

None.
