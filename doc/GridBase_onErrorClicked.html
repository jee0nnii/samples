GridBase onErrorClicked

Description

그리드에 발생한 에러창을 클릭할때 발생한다.

Syntax

function onErrorClicked(grid, error)

Arguments

grid

Type: GridView

GridView 컨트롤

error

Type: String

에러 메시지

Return

None.

Example

grid.onErrorClicked = function (grid, error) {
console.log("onErrorClicked: " + error);
};
