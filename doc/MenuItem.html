MenuItem Class

Properties

label

Type: String

Default: null

메뉴에 표시될 명칭

type

Type: MenuItemType

Default: MenuItemType.NORMAL

메뉴 항목의 형태

group

Type: String

Default: undefined

메뉴의 그룹, type이 radio일때 동일 그룹내에서 하나만 선택

enabled

Type: Boolean

Default: true

사용 여부

checked

Type: Boolean

Default: false

type이 CHECK 또는 RADIO일때 기본 체크여부

tag

Type: Object

Default: null

태그

data

Type: Object

Default: null

children

Type: array of MenuItem

2단계이상일때 하위 메뉴 항목 배열

callback

Type: function (data)

Click callback function
