CopyOptions Class

Properties

singleMode

Type: Boolean

Default: false

true면 선택 영역과 상관없이 포커스를 갖는 셀 하나의 값만 클립보드로 복사한다.

datetimeFormat

Type: String

Default: null

이 값이 지정되면 datetime 필드의 값을 복사할 때 변환 형식으로 사용된다. 지정되지 않은 경우 데이터필드에 설정된 값을 사용한다.

booleanFormat

Type: String

Default: null

이 값이 지정되면 datetime 필드의 값을 복사할 때 변환 형식으로 사용된다. 지정되지 않은 경우 데이터필드에 설정된 값을 사용한다.
