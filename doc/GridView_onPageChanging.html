GridView onPageChanging

Description

그리드가 Paging 상태일 때, 다른 페이지로 이동하기 직전에 호출된다.

Syntax

function onPageChanging (grid, newPage)

Arguments

grid

Type: GridView

GridView 컨트롤

newPage

Type: number

새로 이동하게 될 페이지 인덱스.

Return

Type: boolean

false를 리턴하면 페이지 이동이 취소된다.

Example

grid.onPageChanging =  function (grid, newPage) {
return false;
}
