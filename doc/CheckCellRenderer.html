CheckCellRenderer Class

Check 셀 렌더러는 셀의 값을 true/false 두 가지 상태로 표시하는 렌더러이다

컬럼에 연결된 필드의 자료형이 Boolean이 아니라면, 렌더러의 trueValues에 지정된 값들을 true로, falseValues로 지정된 값들을 false로 판단한다.

properties

trueValues

Type: Array

Default: none

true로 표시할 값들의 배열

falseValues

Type: Array

Default: none

false로 표시할 값들의 배열
