GridBase setPanelOptions

Description

그리드 Panel과 관련된 정보들을 설정한다. Panel이 설정 모델이다.

Syntax

function setPanelOptions(options)

Parameters

options

Type: object

Panel 모델과 같은 설정 정보. Panel 중 변경하고자 하는 속성만 전달하면 된다.

Return value

None.

Example

grid.setPanelOptions({
minHeight: 30,
visible: true,
...
});
