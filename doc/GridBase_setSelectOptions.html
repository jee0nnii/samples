GridBase setSelectOptions

Description

그리드 선택 방식 및 선택 표시와 관련된 정보들을 설정한다. SelectOptions이 설정 모델이다.

Syntax

function setSelectOptions(options)

Parameters

options

Type: object

SelectOptions과 같은 내용의 설정 정보. SelectOptions 중 변경하고자 하는 속성만 전달하면 된다.

Return value

None.

Example

grid.setSelectOptions({
style: "rows"
...
});
