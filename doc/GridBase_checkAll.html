GridBase checkAll

Description

그리드의 데이터행을 모두 체크,해제한다. CheckBar의 exclusive에 영향을 받지않는다.

Syntax

function checkAll(checked, visibleOnly)

Parameters

checked

Type: Boolean

Default: true

visibleOnly

Type: Boolean

Default: false

collapse로 인해 접혀서 보이지 않는 행의 포함여부

Return value

None.

Example

grid.checkAll(true,false);
