GridView groupBy

Description

지정한 필드 목록에 포함된 순서대로 행 그룹핑을 실행한다.

Syntax

function groupBy(fieldNames)

Parameters

fieldNames

Type: Array of String

데이터필드 이름 목록

그룹을 해제하려는 경우 빈 배열을 입력한다.

Return value

None

Example

grid.groupBy(["field1", "field2"]);  // 그룹을 설정하는 경우
grid.groupBy([]);  // 그룹을 해제하려는 경우
