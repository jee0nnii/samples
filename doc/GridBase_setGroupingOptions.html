GridBase setGroupingOptions

Description

그리드 RowGrouping에 관한 설정을 한다. GroupingOptions가 설정 모델이다.

Syntax

function setGroupingOptions(Options)

Parameters

Options

Type: Object

Return value

Type: object

GroupingOptions 모델과 같은 설정 정보. GroupingOptions 중 변경하고자 하는 속성만 전달하면 된다.

Example

var opts = {linear:true};
grdMain.setGroupingOptions(opts);
