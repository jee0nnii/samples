GridView setPage

Description

Paging 모드일 때 표시할 페이지 번호를 지정한다.

Syntax

function setPage(page, startRow)

Parameters

page

Type: number

페이지 번호.

startRow

Type: number

이 페이지에 표시할 첫번째 행의 인덱스. -1이면 setPaging에 지정한 페이지 크기에 따라 결정된다. 지정하지 않으면 -1이다.

Return value

None.

Example

var count = grid.getPageCount();
grid.setPage(count - 1);
