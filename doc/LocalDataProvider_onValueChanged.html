LocalDataProvider onValueChanged

Description

LocalDataProvider.setValue를 이용하여 데이터를 변경한후 호출된다.

Syntax

function onValueChanged(provider, row, field);

Arguments

provider

Type: LocalDataProvider

LocalDataProvider object.

row

Number

수정된 데이터행의 Index

field

Number

수정된 데이터Field의 Index

Return

None.

Example

dataProvider.onValueChanged = function (provider, row, field) {
};
